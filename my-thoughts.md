# My Toughts

---

## Thinking

- ทดสอบดูแล้วคิดว่าการเอาเลย์เอาท์เดสก์ท็อปมาอ้างอิงกับเลย์เอาท์มือถือน่าจะไม่ถูก เพราะถ้าจอแคบมาก อัตราส่วนน่าจะล้น จึงจะแก้ให้พวก padding และ gap แสดงผลแค่เดสก์ท็อปพอ
- เดี๋ยวจะสร้าง `@media` ขึ้นมา แล้วย้ายพวกค่าจำเป็นไปไว้ในนั้นแทน
  - ไม่มั่นใจขนาดจอใหญ่ควรเป็นเท่าไหร่ เดี๋ยวลองด้วย 768px ก่อน
  - ลักษณะของข้อมูลใน nav bar สองเวอร์ชั่นมันต่างกัน อาจจะต้องระบุคลาสเจาะจงเพิ่ม แยกเป็น `.mobile` กับ `.desktop` แล้วใส่ property ให้มัน hidden
- desktop nav ต้องแตกเป็นสอง block ด้วย `div.nav-logo` และ `ul.nav-links` เพื่อที่จะใช้ `space-between` แยกออกไปชิดสองฝั่ง
  - ใน `div.nav-logo` ต้องทำให้เป็น flexbox แล้วจัดเรียงตามสเป็ค
  - ใน `ul.nav-links` ต้องทำให้เป็น flexbox แล้วจัดเรียงตามสเป็ค

---

## Actions

- สร้าง `@media` ขึ้นมาสำหรับปรับสเป็คของ desktop version และ mobile version แยกกัน ด้วย div.xxx
- ลบ `.card` ออก ลดความยุ่งยาก
- ในส่วนของการปรับสเป็ค desktop nav ให้เอไอไกด์ แล้วทำความเข้าใจ เนื่องจากเวลาจำกัด
